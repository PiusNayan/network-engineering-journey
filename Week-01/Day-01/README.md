# Week 01 — Networking Fundamentals

## Overview

This week focuses on building a strong foundation in computer networking. The objective is to understand the basic concepts, terminology, and components that make communication between devices possible before moving on to more advanced networking topics.

---

## Learning Objectives

By the end of Week 1, I will be able to:

* Explain what a computer network is.
* Identify different types of networks.
* Describe the roles of common networking devices.
* Understand how data travels across a network.
* Become familiar with Cisco Packet Tracer.
* Explore the Wireshark interface.
* Develop the habit of documenting networking concepts and practical labs.

---

## Topics Covered

* Introduction to Computer Networking
* Types of Networks
* Networking Devices
* Basic Network Communication
* Introduction to Cisco Packet Tracer
* Introduction to Wireshark
